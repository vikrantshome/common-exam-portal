# GUJCET (Gujarat Common Entrance Test)

## Official Links & Dates (2026)
- **Official Website:** [https://gujcet.gseb.org/](https://gujcet.gseb.org/)
- **Registration Status:** **OPEN**
- **Registration Start:** January 2026
- **Registration End:** February 2026
- **Exam Dates:** April 2026

## Overview
GUJCET is the state-level entrance exam for engineering and pharmacy in Gujarat, conducted by GSEB.

## Data Requirements

### 1. Registration
- Surname, Student Name, Guardian Name.
- Gender.
- Mobile Number.
- Email ID.

### 2. Application Form
- **Personal:**
  - Date of Birth.
  - Caste Category.
  - PwD Status.
  - Family Income.
  - Aadhar Card Number.
- **Academic:**
  - Board Code (e.g., 01-GSEB, 02-CBSE).
  - **School Index Number:** (Unique to Gujarat Board schools).
  - Class 12 School Name.
  - Group: A (Maths) / B (Biology) / AB (Both).
  - Medium: Gujarati / English / Hindi.

### 3. Exam Center
- Select District preference.

- **School Index Number:** Very specific field for GSEB students.
- **Payment:** Often requires buying a "Pin" or paying online to generate a User ID first.
